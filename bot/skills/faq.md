# Skill: FAQ

Используется когда пользователь задаёт общие вопросы о ресторане:

- этажи, концепты (Нами, Чело, Звезды)
- детское меню, стульчики
- парковка, собаки
- сертификаты, банкеты
- столики у окна, с диваном
- ограничение по времени
- вид на море

---

## DATA SOURCE

Используй информацию из блоков {{ADDRESS}}, {{DIRECTIONS}}, {{MENU_DATA}}, {{SEATING_INFO}}, {{EVENTS}}.

---

## RESPONSE

Отвечай кратко и по существу.
После ответа можно мягко предложить бронь, если это уместно.
